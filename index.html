DGA · Guidance, Navigation & Control

Vision-based autonomous interception

A complete guidance, navigation and control system that detects, tracks and intercepts a moving target from a single camera. I built the full estimation-and-control chain: a pinhole projection model, an Extended Kalman Filter for 3D state estimation, and proportional-navigation guidance with a terminal maneuver — validated from simulation to flight.

100% on static targets · moving targets ≤ 5 m/s · 20 Hz control loop · Gazebo · PX4 · ROS 2

Skills
State estimation (EKF) Computer vision Guidance & control laws ROS 2 / Python Sim-to-real
Sample code · estimation-to-guidance command (Python) 
# LOS angle + rate from the EKF state, ego-rotation removed
los_angle = atan2(X, Z)
los_rate  = (Vx*Z - X*Vz)/(X**2 + Z**2) + yaw_rate

# Proportional navigation -> bounded lateral velocity setpoint
Vc    = max(-Vz, forward_speed)
a_lat = N * Vc * los_rate          # N = navigation constant
Vy    = clip((Vy + a_lat*dt)*0.998, -10, 10)
DGA · Multi-agent control

Collaborative formation flight

Control laws that keep several agents in formation along complex trajectories. I developed and compared two architectures — a rigid-body Leader-Follower with PID, and an artificial-potential-field approach — then identified a formation-rotation issue in turns (not covered in the literature) and solved it with a hybrid switching logic. Validated in simulation, then on real hardware in an OptiTrack motion-capture arena.

10-agent scale-up (sim) · ~0.45 m tracking error · sim-to-real validated · OptiTrack
