DGA · Vision-based GNC

Autonomous Drone Interception

A complete Guidance, Navigation & Control system that lets a quadrotor detect, track and physically intercept a moving drone using a single camera — no rangefinder, no GPS lock on the target.

  • Detect — a YOLO network returns a bounding box around the target.
  • Reconstruct — a pinhole camera model turns the 2D box into a 3D direction, using the target's known wingspan to estimate range.
  • Estimate — an Extended Kalman Filter fuses noisy detections into a smooth 3D position + velocity, with range-adaptive noise for terminal stability.
  • Engage — Proportional Navigation flies a collision course; a terminal “uppercut” predicts the impact point and strikes.

100% static targets · moving targets ≤ 5 m/s · 20 Hz loop · Gazebo · PX4 · ROS 2

Sample code · Proportional Navigation lateral command (Python) 
# LOS angle + rate from the EKF state, ego-rotation removed
los_angle = atan2(X, Z)
los_rate  = (Vx*Z - X*Vz)/(X**2 + Z**2) + yaw_rate

# ProNav lateral acceleration -> bounded velocity setpoint
Vc    = max(-Vz, forward_speed)
a_lat = N * Vc * los_rate          # N = navigation constant
Vy    = clip((Vy + a_lat*dt)*0.998, -10, 10)
DGA · Multi-agent control

Collaborative & Swarm Flight

Control laws that keep a swarm of quadrotors in formation while following complex trajectories — ellipses and rounded-square Bézier paths, with altitude variation.

  • Leader-Follower — treats the formation as a rigid body; kinematic feedforward + PID for high geometric precision.
  • GS-RUFF — the formation shape emerges from attraction/repulsion forces between agents, giving intrinsic collision avoidance.
  • Hybrid fix — I identified and solved a formation-rotation problem in turns (not covered in the literature) with a switching logic.

Validated in Gazebo, then transferred to real drones in an OptiTrack motion-capture arena (millimeter-precision localization via VRPN → ROS 2 → MAVLink → PX4). Single-drone tracking matched simulation closely.

10-drone scale-up (sim) · ~0.45 m tracking error · OptiTrack localization
